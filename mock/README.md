# ローカル環境での起動手順

## パッケージインストール

```
$ npm install
```

## 起動

```
$ npm start
```

# 画面一覧

| 画面名       | URL                                                   |
| :----------- | :---------------------------------------------------- |
| テナント登録 | http://localhost:3001/tenant                          |
| ポータル     | http://localhost:3001 or http://localhost:3001/portal |
